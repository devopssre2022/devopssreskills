Command Line Instructions: Setting up repo locally
-------------------------
$ git config --global user.name "enter_name"
$ git config --global user.email "enter mail"

Create New Repo:
---------------
$ git clone git_url......
$ cd my_project
$ touch README.md
$ git add .
$ git commit -m "New File Created"
$ git push -u origin master/main

Push an existing folder:
---------------------
cd existing folder
git init
git remote add origin git_url.....
git add .
git commit -m "Initial Commit"
git push -u origin master

Push to existing Repo:
---------------------
cd existing folder
git remote rename origin old-origin
git remote add origin git_url....
git push -u origin --all
git push -u origin --tags

git fetch  - fetches the changes but wont pull code

git status - to check the status of files in working directory

git commit  - to commit changes to local repo

git push - to push the commit to central repo
